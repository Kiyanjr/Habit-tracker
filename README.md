# Habit Tracker

Habit Tracker is a simple productivity app built with Flutter that helps users build and maintain positive habits.  
The main goal of this project is to provide an easy way to track daily routines and stay consistent over time.

This project was created as a learning project to practice Flutter development, UI design, and app structure.

## âš™ï¸ How the App Works

The workflow of the application is simple:

1. User creates a new habit.
2. The habit appears in the habit list.
3. Each day the user can mark the habit as completed.
4. The app tracks the completion progress.
5. Over time, users can see how consistent they are.

This helps users stay motivated and organized.
